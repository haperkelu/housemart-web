好识买网络  《登盘使用手册》

1. 进入官网

在浏览器地址栏输入：http://www.housemart.cn  (目前我们只支持IE8以上版本浏览器，以及Firefox浏览器或者谷歌浏览器。建议使用谷歌浏览器获得最佳体验。)

 

2.点击“经纪人服务”

3.点击“去登盘”

 

 

4. 登录

输入用户名和密码并登录。例如用户名：mo.jia  密码：******

5. 进入"我的房源"

点击发布售房按钮->进行房源发布

 

 

6.发布售房-第一步-填写基本信息

 

7.发布售房-第二步-上传图片

  《房源图片》

《房型图片》

《小区图片》

8. 发布售房-第三步-完成

这时候房源就提交申请了，

8.1 审核中的房源列在《审核中》栏。在《审核中》的房源，你可以《修改》该房源信息。也就是重复上面第6，第7步骤。

8.2 被拒绝的房源会出现在《拒绝》栏。这时候，你可以修改房源信息，然后再次申请《上架》。

8.3 通过审核的房源就会出现在《上架房源》栏。5分钟后，房源会在手机端展示，此时你不能修改房源信息。如果想修改房源信息，需要先《下架》房源。下架后房源会进入《下架房源》栏，此时你可以修改房源信息，然后提交上架申请。

8.4 你自己也可以下架自己的房源，下架的房源出现在《下架房源》栏。下架的房源你可以《修改》信息，或提交《上架》申请。
